Перед запуском ввести в консоль ---> npm install <--- Для установки всех зависимостей.

Для запуска проекта ---> npm start

Если будет спрашивать о файлах jsx, то он позволяет HTML подомный код использовать внутри JavaScript + Общество разработчиков на React им активно пользуются. Также он прост в применение.
Это тот же JavaScript поэтому ничего нового учить не надо.

В файле уже созданы некоторые файлы, которые можете уже начать редактировать под макет.

Пути для проверки работы проекта:

http://localhost:8080/
http://localhost:8080/content
http://localhost:8080/login
http://localhost:8080/register
http://localhost:8080/cabinet/(client|admin|landlord)
